# daft-geo

Geospatial extension for [Daft](https://github.com/Eventual-Inc/Daft), built with Rust (PyO3) and Arrow.

## Installation

```bash
uv add daft-geo
```

## Development

```bash
# Install in development mode
uv sync --extra test

# Run tests
pytest tests/
```
